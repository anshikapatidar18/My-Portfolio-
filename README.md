📧 EmailJS Setup

To make the contact form work:

- Replace YOUR_EMAILJS_PUBLIC_KEY
- Replace YOUR_SERVICE_ID
- Replace YOUR_TEMPLATE_ID

inside script.js with your own EmailJS credentials.
